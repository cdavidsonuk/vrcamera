# GitHub Pages upload instructions

1. Open the GitHub repository.
2. Delete or replace the old `index.html`, `assets`, and `src`.
3. Upload the contents of this package.
4. Commit the changes.
5. Open **Settings → Pages**.
6. Confirm the site publishes from the correct branch and `/root`.
7. Wait one or two minutes, then refresh the Pages URL using Ctrl+F5.
