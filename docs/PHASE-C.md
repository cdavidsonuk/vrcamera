# Phase C — Interactive Camera

## Delivered
- Phase B artwork and independent control layers integrated into one camera application
- Camera power and boot sequence
- Rotating mode dial and rear command wheel
- Two-stage autofocus/capture shutter behaviour
- Movable focus point and AF-ON confirmation
- Live shooting LCD
- Main menu and menu selection
- Quick settings screen
- Playback and delete confirmation
- INFO display toggle
- Stable camera state engine
- Keyboard, pointer and responsive controls
- Capture records containing mode, aperture, shutter speed and ISO

## Controls
- P: power
- M: mode dial
- Left/right arrows: rear wheel
- Arrow keys: move focus point
- Space: autofocus then capture
- Escape: return to shooting screen

## Scope boundary
Phase C implements the camera interaction and operating system. It does not yet render photographic depth of field, motion blur, noise or white balance changes. Those belong to the exposure and scene engine phase.
